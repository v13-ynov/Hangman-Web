Régles du pendu :

Bienvenue au jeu du pendu. Avant de commencer, voici quelques règles :

But du jeu :

- Le but est de trouver le mot en soumettant diverses lettres ou en devinant le mot directement ( avec les accents si besoins.)

- Si une lettre est incorrecte, alors le nombre de tentatives diminue.

- Si vous devinez un mot incorrect, alors vous perdez 2 tentatives.

- A partir de 10 tentatives qui sont perdues, alors vous avez perdu.

- Si vous arrivez à deviner le mot, vous avez gagné.

Mode Difficile :

- Si vous utilisez une même lettre alors une tentative en moins.

- Les lettres que vous avez utilisées ne seront plus affichées sur "Lettres devinées".

- Pénalité de 1 tentative si vous utilisez plus de 3 voyelles ( peu importe si la voyelle est correct ou non).

- Moins de lettres seront affichées au début du jeu.
